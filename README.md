# Two-Wheeler-Sale-Purchase-System

🚀 Two Wheeler Sale Purchase System – Project Showcase (Tkinter + MySQL)

📅 Completed: Dec 2025

I’m excited to share one of my recent projects — a Two Wheeler Sale Purchase System, built using Python Tkinter for the GUI and MySQL as the backend database.
This project is designed to simplify and automate the workflow of two-wheeler dealerships, making operations faster, reliable, and error-free.

🔧 Tech Stack
Python (Tkinter) – GUI development
MySQL – Database management
FPDF / ReportLab – PDF bill generation
Pillow (PIL) – Image handling
OS / Threading – Smooth operations & file processing

📌 Key Features

1️⃣ Customer Management
Add, update, search, and delete customer records
Real-time validation to avoid duplicate data
Customer dropdown auto-filled from database

2️⃣ Vehicle Model Management
Store and manage all vehicle models
Track model specifications like engine, chassis, color, battery etc.
Auto-fetch model data during billing

3️⃣ Billing System (Main Feature)
Complete billing interface with:
Bill number auto-generation
Customer selection
Model selection
Real-time price fetching
Auto calculations and validation
Instant PDF bill generation ("Print Bill")

4️⃣ PDF Bill Generation
Professional invoice layout with:
Dealer logo
Customer details
Vehicle details
Pricing
Remarks
Automatically opens after creation
Ensures clean, uniform bills for customers

5️⃣ Smooth & Clean UI
Modern Tkinter UI design
Color-coded buttons
Status bar for live messages
Full-screen responsive dashboard
Reduces manual work & improves dealership workflow

🎯 Project Outcomes
✔ Automated the entire sale-purchase workflow
✔ Reduced manual entry errors by introducing field validation
✔ Generated clear, professional PDF bills
✔ Improved data retrieval speed using MySQL
✔ Delivered a highly user-friendly interface

📘 Learning Highlights
This project helped me strengthen:
GUI development using Tkinter
Database connectivity using MySQL Connector
Real-time form validation
Object-oriented design in Python
Working with PDF libraries
Modular coding and multi-file project structuring

🛠 Use Cases
This system can be used in:
Two-wheeler showrooms
Small vehicle resellers
Garage-level vehicle sales
Any shop generating manual bills
